# 📱ARIS-M
Angular Ranging Integrated System - Mobile
![サンプル画像](ARIS-M/images/sample.png)  
## 概要
ARIS-Mはスマートフォンやタブレット等で動作する簡易高度計測ツールです。任意のベースライン（射点-観測点までの距離）を入力し、端末をロケットに向けて計測を実行します。  

ARIS-Mでは計測点が一つとなるシングルステーショントラッキング法を想定します。  
この方法には角度による誤差が乗り、風の流され量を考慮できないというデメリットがあるため  
1\.なるべく離れた位置から[^eins]  
2\.風向きに対して直交する方向から計測する  
事を推奨します。  
![構成図](ARIS-M/images/overview_diagram.png)  
2名での計測が行える場合は、両者の高度結果の平均値を出すことで全国大会での計算法に近づけることができます。[^zwei]

## 計測について
![大体のイメージ](ARIS-M/images/usage_example.png)  
### センサの扱い
計算に使用する仰角は端末のpitch角を読み取り、使用します。  
ただし、その数値は端末を机に置いた時（画面を空に向けた状態）に0°となる為、本ツールではpitch角のみ+90をし、端末を垂直に取った時に0°となるように調整を行っています。

### 計算式
高度は以下のように計算されます  

`h`: 高度 (m)  
`bl`: ベースライン (m)  
`θ`: 基準状態からの仰角 (deg) 
```math
h = bl * tan(θ)
```
※ツール下部に表示される式ではコードをそのまま表記しているため一度rad変換が入っています。

#### 高度誤差量の算出 
画面内「MaxDev」に表示される高度誤差は以下のように計算されます  
`MaxDev`: 高度誤差量 (m)  
`AE`: 角度誤差量 (deg)
```math
MaxDev = h - (bl * tan(θ + AE))
```

④ リアルタイム表示を行います。高度誤差量がアラートレベルを超えた場合、赤く表示されます。  
⑤ ベースラインを表示します。  
⑥ 推定高度のリアルタイム表示を行います  
⑦ 高度誤差量を表示します  
⑧ タップでカメラの接続を行います  
⑨ タップでセンサの接続を行います  
⑩ 計測を実行します  
⑪ ベースラインの入力エリアです  
⑫ 実行結果が表示されます  
⑬ アラートレベルを設定できます。単位はmです  
⑭ 角度誤差量を設定できます。単位はdegです  
⑮ 端末の姿勢角を表示します  
⑯ キャリブレーション値を表示します  
⑰ タップで実行結果をCSVデータとしてダウンロードします

### 計測を行う
 1. 下記リンクにアクセスします。もしくはリポジトリ内の必要ファイルをダウンロードし、実行してください。  
 2. センサ（⑨）をタップし、センサのアクセスを許可してください。  
 3. ベースラインの入力エリア（⑪）にベースラインを数値のみ入力してください。単位はメートルです。  
 4. カメラ（⑧）をタップし、カメラのアクセスを許可すると画面にカメラ映像と照準を表示することができます（こちらは任意です）。  
 5. ロケットの最高高度位置に端末を向け、計測ボタン（⑩）をタップして計測します。  
 6. 実行結果が画面下部（⑫）に表記されます。  
#### ⚠注意⚠
- 計測の際は可能な限り端末の中心を軸にし、端末を傾けてください。通常の動画撮影のように腕を伸ばした状態で傾けると誤差が発生します。
- ARIS-Mでは風による流され量を補正できません。単体で補正を行う場合は最高高度でのベースラインを別途測定し、再計算を行ってください。なお、大会ではこのような補正は行われません。

### アラートレベルの設定
仰角が上がるにつれて角度にずれが生じたときの結果に与える誤差量が大きくなります。  
そこでARIS-Mではアラートレベルを設定し、設定値を超えたときに警告としてリアルタイム表示部を赤く表示します。  
アラートレベルはオプションタブ上部の二値によって設定されます。  
`AL`: アラートレベル (m) / 算出誤差がこの値を超えたときに表示が切り替わります。デフォルトは10mです。  
`AE`: 角度誤差量 (deg) / 想定される角度誤差です。デフォルトでは1°です。

### キャリブレーションについて
姿勢角を補正できます。端末を正しい角度に保持した状態でオプションタブ内のキャリブレーションボタン⑱をタップすると反映されます。  
なお、この値はツール内部で読み取った値を補正するのみにとどまり、本体のセンサには影響しません。この値はページのリロードでリセットされます。

### 計測データのダウンロード
計測データをCSV形式でダウンロードできます。オプションタブ下部の`CSVデータダウンロード`⑰をタップすると保存先を聞かれるのでお好きな場所に保存してください。
データは次のように記録されます
| date     | time     | Roll         | Yaw         | Pitch       | CRoll        | CYaw        | CPitch        | FPitch        | BL           | TAlti       | 
| -------- | -------- | ------------ | ----------- | ----------- | ------------ | ----------- | ------------- | ------------- | ------------ | ----------- | 
| 日付     | 時刻     | roll角       | yaw角       | pitch角     | roll角補正値 | yaw角補正値 | pitch角補正値 | 補正後pitch角 | ベースライン | 高度        | 
| 出力例↓ |          |              |             |             |              |             |               |               |              |             | 
| 52       | 17:24:35 | -2.010242547 | 39.99220162 | 49.08557556 | 0            | 0           | 0             | 49.08557556   | 30           | 34.58497336 |
