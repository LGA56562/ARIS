
# ARIS
Angular Ranging Integrated System

## 概要
ARISは仰角からの高度推定を補助するためのツール集であり、モデルロケットでの利用を主として製作しております。

## 📱ARIS-Mobile
仰角による高度計測をスマートフォンを使用して手軽に行うためのツールです。
HTMLで動作するため、スマートフォンやタブレットといったwebブラウザが開けるジャイロセンサの類が搭載されているデバイスで動作させることができます。

[詳細はこちらから](https://github.com/LGA56562/ARIS/tree/main/ARIS-M "ARIS-M")
